# CORE – Frontend Practical Assignment

This project is a pixel-perfect implementation of the provided Figma design using **React + Vite**.  
The UI has been carefully developed to match the layout, spacing, typography, colors, and responsiveness specified in the design.

The application includes:
- Sidebar navigation
- Header with MST timezone and live clock
- Search functionality
- Employee grid view
- Responsive layout (Desktop + Mobile)
